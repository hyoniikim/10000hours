# 1만 시간의 법칙 반응형 웹 구현 프로젝트

## 💖 프로젝트 소개

HTML와 CSS를 이용하여 반응형 웹 구현하였습니다.

## 🧡 개발 기간

- 2024.07.01 ~ 2024.07.02

## 💛 멤버구성

- 김효원 (개인)

## 구현내용

- "훈련하러 가기 GO!GO!" 버튼을 클릭하면 모달창 열림
- 모달창에서 "종료하고 진짜 훈련하러 가기 GO!GO!" 버튼을 클릭하면 모달창 닫힘
